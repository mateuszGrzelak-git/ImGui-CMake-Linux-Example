# ImGui-CMake-Linux-Example
Dependencies: <br />
git clone --recursive https://github.com/mateuszGrzelak-git/ImGui-CMake-Linux-Example <br />
git submodule update --init   <br />
sudo dnf install glfw-devel mesa-libGL-devel <br />

How to Install: <br />
mkdir build <br />
cd build <br />
cmake .. <br />
cmake --build . <br />
